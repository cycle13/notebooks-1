#### 引言

在很长一段时间内，都要在地球流体动力学领域学习、编程、建模、画图……今年开始学习[Python](https://www.python.org/)，所以这里新建了一个[repo](https://github.com/miniufo/notebooks)，用于展示自己学习[Python](https://www.python.org/)、并应用于地流领域的一些成果和心得。

之前一直用[cvs](https://www.cvs.com/international.html)管理版本，后来发现[github](https://github.com/)吊打[cvs](https://www.cvs.com/international.html)（我只用到皮毛，也没真体会到），就换了。结合[eclipse](https://www.eclipse.org/)用了一段时间，大约2年，但是没有系统学习过、也没用过命令行进行git的操作。现在借着这个机会，系统学习一下这个工具，毕竟要加入全世界的软件开发浪潮，git是必备技能。

十多年前之前写过一个[博客](http://miniufo.bokee.com/)，也分享一些学习心得，后来不是懒，是受到点打击（long-story），没坚持下去，浪费了十多年时间去积累。But never late than never。现在正是开始的时候。
